StrategyV13 – Sizing & Stops (Quick Toggle Guide)

Current mode (this file):
- Risk sizing: ON
  - Risk per trade = account_balance * risk_pct_of_balance
  - Size = floor( risk_dollars / R_per_share ), clamped to [min_size, max_size], rounded to lot_size
  - R_per_share = |entry - stop|, stop currently via ATR (atr_mult)
- Stops: ATR (stop_mode="atr", atr_mult=2.0)
- Partials: cumulative-of-original (25%, 25%, 50%) with base_size only used when risk sizing is OFF

To match the old script (fixed base size, same partials):
- Set: "use_risk_sizing": false
- Ensure: "base_size": 4
- Keep: "stop_mode": "atr", "atr_mult": 2.0
This replicates legacy behavior (partials 1, 1, 2 → flat).

If you prefer fixed % stops instead of ATR:
- Set: "stop_mode": "fixed"
- Add: "fixed_sl_pct": 0.03   (example: 3%; use 0.05 for 5%)
- Keep a reasonable "max_size" if risk sizing is ON.

A/B testing tip:
- Save two files:
  - configs/strategy_v13_risk_atr.json  (risk ON, ATR)
  - configs/strategy_v13_base_atr.json  (risk OFF, base_size=4, ATR)
- Run backtests by swapping --config.
